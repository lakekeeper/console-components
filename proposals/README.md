# Feature Proposals

This folder contains community feature proposals for the Lakekeeper console.

## How to propose a feature

1. Copy the [FEATURE_PROPOSAL.md](../FEATURE_PROPOSAL.md) template
2. Save it here as `proposals/your-feature-name.md`
3. Fill in the sections
4. Open a PR

Accepted proposals will be added to the [ROADMAP.md](../ROADMAP.md).
